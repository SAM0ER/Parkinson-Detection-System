# Parkinson's Detection System 🧠🎤

This project is a diagnostic tool designed to detect **Parkinson's Disease** through voice features analysis. 

### 🚀 Features
- **Live Recording:** Real-time audio capture and processing.
- **Machine Learning:** Built using Scikit-learn and specialized voice extraction libraries.
- **Clinical Focus:** Designed as part of the Clinical Engineering curriculum at **MTI University**.

### 🛠️ Technology Stack
- **Language:** Python 3.10
- **Framework:** Streamlit
- **Libraries:** Librosa, Pydub, Scikit-learn, Parselmouth.
